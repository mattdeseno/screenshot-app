========================================
NEXT STEPS (5 minutes)
========================================

1. Deploy Backend to Leapcell:
   - Go to https://leapcell.io/workspace
   - Click "+ Create Service"
   - Select "Deploy from GitHub"
   - Choose "mattdeseno/screenshot-app"
   - Configure:
     * Root Directory: backend
     * Build Command: npm install && npx playwright install chromium
     * Start Command: node server.js
     * Port: 4000 (or 8080 if required)
   - Click "Deploy"

2. Get Backend URL:
   - Copy your Leapcell service URL
   - Example: https://screenshot-app-xxxxx.leapcell.app

3. Update Frontend:
   - Edit index.html on GitHub
   - Replace 'REPLACE_WITH_YOUR_LEAPCELL_URL/screenshot'
   - With your actual Leapcell URL
   - Commit changes

4. Test:
   - Visit https://mattdeseno.github.io/screenshot-app/
   - Enter any URL (e.g., https://www.google.com)
   - Click "Generate Screenshot"
   - Done!
